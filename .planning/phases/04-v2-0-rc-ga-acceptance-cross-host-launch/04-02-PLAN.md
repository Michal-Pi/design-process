    AXE-RUNNER CORE (assets/scripts/axe-runner.mjs):

    exports: runAxeOnFixture, buildTokenScaffold, runAxeRunner

    buildTokenScaffold(tokens):
    Walk the DTCG tokens object recursively. For each token with $type === 'color' and a $value string:
      1. Parse the OKLCH value with culori.parse(value)
      2. Convert to hex: culori.formatHex(parsed)
      3. Emit CSS custom property: --token-<tokenPath>: <hex>
    Build HTML: '<html><head><style>:root { <custom-props> }</style></head><body>' + colored divs for each color token + '</body></html>'
    Each div: '<div style="background-color: var(--token-<path>); color: #000; padding: 8px; font-size: 16px;">Sample text for ' + tokenPath + '</div>'
    This ensures axe-core has rendered DOM with contrast-checkable text.

    runAxeOnFixture(fixtureDir):
    1. Load tokens.json from path.join(fixtureDir, 'expected', 'tokens.json'). If absent, return { fixtureId, pass: true, violations: [], contrastValues: {}, note: 'no tokens.json, skipped' }
    2. Build scaffold via buildTokenScaffold(tokens)
    3. Launch Playwright chromium browser (use chromium.launch({ headless: true }))
    4. await page.setContent(htmlScaffold)
    5. await page.addScriptTag({ content: axeSource }) — where axeSource is readFileSync(require.resolve('axe-core')) or import.meta.resolve('axe-core')
    6. const results = await page.evaluate(async () => window.axe.run({ runOnly: { type: 'tag', values: ['wcag2aa'] } }))
    7. Filter for id 'color-contrast' violations
    8. Close browser after each fixture (do not share browser instance — avoid test isolation bleed)
    9. Return { fixtureId, pass: results.violations.length === 0, violations: results.violations, contrastValues: extractContrastValues(results) }

    runAxeRunner(opts):
    1. Path-traversal check: resolved = path.resolve(opts.fixturesDir); if (!resolved.startsWith(projectRoot)) throw Error('path-traversal')
    2. Load fixtures.manifest.json
    3. Run runAxeOnFixture for each fixture (sequential to avoid headless browser concurrency issues)
    4. Collect results
    5. hasViolations = results.some(r => !r.pass && !r.note?.includes('skipped'))
    6. Write axe-results.json with: { pass: !hasViolations, fixtureCount: 15, passingFixtures: [...], failingFixtures: [...] } — both count AND identities (Lesson 5)
