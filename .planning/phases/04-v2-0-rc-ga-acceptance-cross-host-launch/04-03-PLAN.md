    selectDeterministicSample(fixtures: FixtureManifestEntry[], sampleSize: number): FixtureManifestEntry[]
      - input: 15 fixtures (5 B2B SaaS + 5 consumer + 3 dashboard + 2 marketing), sampleSize=5
      - output: deterministic 5-item sample: 1 b2b-saas + 1 consumer + 1 dashboard + 1 marketing + 1 route-mandatory (mature-app-refactor or DS-extraction)

    computeParityDelta(hostPassRate: number, baselinePassRate: number): number
      - input: 0.6, 0.8 → 0.2
      - input: 0.8, 0.8 → 0.0
      - input: 1.0, 0.9 → 0.1

    runCrossHostParity({ host, sample, baseline, output }): Promise<ParityResult>
      - Loads fixtures.manifest.json
      - Selects deterministic sample of opts.sample (default 5) fixtures
      - For each sampled fixture: sets HOST_PROFILE=opts.host env, runs dispatchRoute() + all 6 gate checks
      - Computes host pass-rate; loads baseline from opts.baseline (JSON file path)
      - Computes delta = |host_pass_rate - baseline_pass_rate|
      - If delta > 0.10 AND sample < 15: escalates to full N=15 re-run on that host
      - If still > 0.10 after escalation: exits 1 (DIST-05/06 failure)
      - Writes parity-results.json with { host, sampleSize, sampledFixtures: [fixtureId...], hostPassRate, baselinePassRate, delta, escalated, pass }
      - Both count (pass-rate) AND identity (sampledFixtures list by fixtureId) per Lesson 5
  </behavior>

    CROSS-HOST PARITY CORE (assets/scripts/cross-host-parity.mjs):

    exports: selectDeterministicSample, computeParityDelta, runCrossHostParity

    selectDeterministicSample(fixtures, sampleSize=5):
    Group fixtures by useCase. Take fixture[0] from each of: 'b2b-saas', 'consumer', 'dashboard', 'marketing' (4 fixtures). For the 5th: take the first fixture where route === 'mature-app-refactor' || route === 'DS-extraction'. Total = 5. If any category is missing fixtures (marketing has only 2), use the first available. Return array is always the same order for the same input.

    runCrossHostParity(opts):
    1. Path-traversal: if opts.fixturesDir is provided, resolve + projectRoot prefix check (Lesson 7)
    2. Load fixtures.manifest.json from opts.fixturesDir or 'evals/acceptance'
    3. Load baseline pass-rate from opts.baseline (JSON file: { claude-code: { passRate: 0.87, ... } }). If baseline not provided, use 0.0 (pessimistic: will escalate to full run)
    4. sample = selectDeterministicSample(fixtures, opts.sample ?? 5)
    5. For each sampled fixture:
       a. Set process.env.HOST_PROFILE = opts.host (codex-cli or cursor)
       b. dispatchRoute(fixture.route, prdPath, { tokenBudget: fixture.budgetCeiling })
       c. Run all 6 gates via runGate() against staged output path
       d. fixture passes if all gates in ['pass', 'pass_with_warnings']
    6. hostPassRate = passCount / sample.length
    7. delta = Math.abs(hostPassRate - baselinePassRate)
    8. if (delta > 0.10 && opts.sample < 15):
         escalate: re-run with full 15 fixtures
         re-compute hostPassRate + delta
    9. Write parity-results.json: { host, sampleSize: used_sample_size, sampledFixtures: [fixtureId...], hostPassRate, baselinePassRate, delta, escalated: boolean, pass: delta <= 0.10 }
    10. process.exit(delta <= 0.10 ? 0 : 1)
