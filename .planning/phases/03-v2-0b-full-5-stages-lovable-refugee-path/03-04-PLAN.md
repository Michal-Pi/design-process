<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| --source path input | User-provided path to a local repo or live URL. Path traversal and out-of-scope access risks. |
| Playwright URL crawler | Fetches and writes assets from a live URL to local temp dir. Content injection risk. |
| design/inferred/ → design/ promotion | User-controlled file copy pathway. Must block inferred artifacts from entering committed design/. |
| migration --apply | Writes over existing design/ia/sitemap.json and persona files. Data loss risk if applied twice. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-03-04-01 | Tampering | --source path traversal | mitigate | Validate source path is inside process.cwd() using path.resolve(). Reject paths containing '..'. |
| T-03-04-02 | Information Disclosure | Playwright URL crawler accessing /api/ endpoints | mitigate | Exclude paths containing '/api/', '/auth/', '/admin/', '.env' from asset discovery (OQ-5 + RESEARCH.md security). |
| T-03-04-03 | Tampering | promote-inferred overwrites clean design/ artifact | mitigate | promote-inferred checks for provenance:inferred AND INFERRED banner still present BEFORE blocking. User must explicitly clean both. |
