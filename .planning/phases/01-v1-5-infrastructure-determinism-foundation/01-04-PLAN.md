<objective>
Land the `design/` directory governance layer: `.gitignore` / `.gitattributes` defaults, the `complete-design init` writer that installs them into a target repo, the PII scanner with the regex + Luhn set + allowlist + pre-commit hook installation, the MANIFEST.md auto-reconciler (ART-07), the recovery prompt wrapper that makes Plan 03's `recover.mjs` interactive (PERSIST-04), the override-banner propagation helper (D-11's downstream artifact mechanism), the SPINE linearity static check (SPINE-04), and the 3 SKILL.md skeletons for complete-design' v1.5 v2.0a-bound skills (`design`, `audit`, `handoff`) carrying agentskills.io v1-compliant frontmatter (DIST-01 — the spec compliance level; DIST-02 + DIST-03 trigger-budget / count enforcement are Plan 05). Also ship the binding TRUST-01..05 trust-posture document and the copy-review checklist for marketing / README / SKILL.md description text (TRUST-04 — never lead with "AI design" framing).

Purpose: Without this plan, complete-design outputs are unprotected against committed-PII leaks, MANIFEST.md drifts from the filesystem state, overrides have no audit propagation, and the trust posture lives only in the MRD instead of in the code/docs. Plan 04 also closes 4 of the 7 Phase-1-prevented pitfalls (Pitfall D — PII scanner false positives; the line-start anchored transcript-header regex prevents the "Interviewer" noun-phrase false-positive; Pitfall I — PII allowlist with content-hash drift; and the SPINE-04 enforcement closes the "downstream stage mutates upstream" anti-pattern).

CLI integration: Plan 04 does NOT modify `bin/complete-design.mjs`. Per Plan 01's auto-discovery dispatcher pattern, Plan 04 contributes new subcommand modules under `assets/scripts/cli/` (one file per subcommand, exporting `{ name, describe, builder, handler }`). The dispatcher auto-loads them at startup. This eliminates the Wave-3 contention on `bin/complete-design.mjs` flagged by the checker.

Output: Init CLI + `.gitignore` / `.gitattributes` templates + PII scanner + allowlist support + pre-commit hook installer + MANIFEST.md reconciler + recovery prompt wrapper + override banner propagator + SPINE linearity check + 3 SKILL.md skeletons + TRUST-POSTURE.md + COPY-REVIEW-CHECKLIST.md + 7 new CLI subcommand modules under `assets/scripts/cli/` + comprehensive governance tests.
</objective>
