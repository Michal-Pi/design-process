Land the `design/` directory governance layer: `.gitignore` / `.gitattributes` defaults, the `design-os init` writer that installs them into a target repo, the PII scanner with the regex + Luhn set + allowlist + pre-commit hook installation, the MANIFEST.md auto-reconciler (ART-07), the recovery prompt wrapper that makes Plan 03's `recover.mjs` interactive (PERSIST-04), the override-banner propagation helper (D-11's downstream artifact mechanism), the SPINE linearity static check (SPINE-04), and the 3 SKILL.md skeletons for design-os' v1.5 v2.0a-bound skills (`design`, `audit`, `handoff`) carrying agentskills.io v1-compliant frontmatter (DIST-01 — the spec compliance level; DIST-02 + DIST-03 trigger-budget / count enforcement are Plan 05).
