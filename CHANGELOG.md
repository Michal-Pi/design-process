### Known Cost Behavior

Sequential-fallback wall-clock caveat: when Claude Code is unavailable, complete-design falls back
to Codex CLI or Cursor with sequential (non-parallel) stage dispatch. This extends wall-clock
time beyond the p50 ‚â§8 min target measured on Claude Code. See `RELEASE-NOTES.md` for the
full cost disclosure generated by `writeReleaseNotesDisclosure()`.

COST-07 and COST-10 will be measured after Wave B Step 0 (npm @latest publish + real LLM
dispatch via `npm run release-gate`).
